# AI-Powered Resume Builder

This is a full-stack application that generates customized resumes based on job descriptions and user skills.

## Features

- Frontend interface for entering profile details, skills, projects, and job descriptions
- Backend APIs integrated with AI models for resume generation
- Database storage for user profiles and resumes
- Resume preview and PDF export

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
- MongoDB
- OpenAI API
- jsPDF for PDF export
